# 组合总和||

https://leetcode.cn/problems/combination-sum-ii/



特点：

1. 给定集合选取元素，集合内元素会重复（需要排序）
2. 每个数字在每个组合中只能使用一次（不能直接对原集合去重）
3. 解集不能包含重复的组合（难，可以利用之前做过的题来去重：三数之和和四数之和



```c++
class Solution {
private:
    vector<vector<int>> ans;
    vector<int> path;
    void backtracing(vector<int>& candidates, int target, int sum, int start){
        if(sum == target){
            ans.push_back(path);
            return;
        }else if(sum > target) return;

        for(int i = start; i < candidates.size(); i++){
          	// 去重后一个数和前一个一样就不再重复
            if(i > start && candidates[i] == candidates[i-1]) continue; 
            path.push_back(candidates[i]);
            backtracing(candidates, target, sum+candidates[i], i+1);
            path.pop_back();
        }
    }
public:
    vector<vector<int>> combinationSum2(vector<int>& candidates, int target) {
        sort(candidates.begin(), candidates.end());
        // auto last = unique(candidates.begin(), candidates.end());
        // candidates.erase(last, candidates.end()); // 上面两行可以用来给排好序的 vector 进行去重
        backtracing(candidates, target, 0, 0);
        return ans;
    }
};
```
